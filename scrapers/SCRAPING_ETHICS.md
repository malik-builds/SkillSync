# SkillSync Scraping Policy

## Source: topjobs.lk
- Data used: Job title, company name, required skills only
- Data NOT stored: Salary, contact details, personal information
- Request rate: Maximum 1 request per 2 seconds
- User agent: Clearly identified as academic research bot
- Robots.txt: Check and respect before each scrape
- Data retention: Jobs marked inactive after 30 days
- Purpose: Academic project - IIT Sri Lanka, not for commercial use
